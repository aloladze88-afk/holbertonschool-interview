# Lockboxes

This project contains a Python function that checks whether all locked boxes can be opened.

Each box may contain keys to other boxes. Box `0` is already open. The function follows the available keys and returns `True` if every box can be opened, otherwise it returns `False`.

## File

- `0-lockboxes.py`: contains the `canUnlockAll(boxes)` function.

## Concept

This task practises basic graph traversal using lists, sets, and loops.
